Rules of Variables:-

1) Variables Should not contain keywords:-

int,float x

2) Variables should not contain white spaces:-

no of balls x
no_of_balls

3) Variables do not start with numbers:-

1num x
num1

4) Variables must start with a-z or A-Z.
